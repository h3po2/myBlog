有关footer的解决方案

2018年3月22日

当页面内容较少无法撑起窗口高度导致footer上浮，这是需要解决的问题，另外若还要根据 页面的内容高度动态调整footer的位置，下面是一个解决方案。

html文档结构是这样的：

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>footer的解决方案</title>
<link type="text/css" media="screen" href="footer.css" rel="stylesheet"></link>
</head>
<body>
<div id="allcontent>
<div>
main
</div>
<footer>
<p>
content of footer.
</p>
</footer>
</div>
</body>
</html>

先设置html和body的高度为100%；

html, body {
height: 100%;
}

将包裹body内所有内容的容器allcontent的定位设为relative， 并设置最小高度为100%（使其与html和body的高度一致），高度设置为自动。

#allcontent {
position: relative;
height: auto;
min-height: 100%;
}

将footer的定位设置为absolute，bottom设置为0px，将其固定在allcontent的下端。

footer {
position: absolute;
bottom: 0px;
}
