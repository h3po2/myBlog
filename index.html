Ubuntu已知问题集锦

2017年12月7日

0 Nvidia显卡安装

sudo apt-get install nvidia-384

1 亮度调节问题

修改/etc/default/grub：

sudo gedit /etc/default/grub
将其中的
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"
改为
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash acpi_backlight=vendor"
然后更新grub：
sudo update-grub
重启后在/sys/class/backlight/下多了一个ideapad的文件夹， /sys/class/backlight/ideapad目录下的brightness和actual_brightness保存 了当前屏幕亮度值，max_brightness是屏幕最大亮度。此时即可调节亮度。

2 关机硬盘异响

修改/etc/init.d/halt的文件：

sudo gedit /etc/init.d/halt
里面有个stop函数，有几行如下
log_action_msg "Will now halt"
halt -d -f $netdown $poweroff $hddown
将其改为
umount -a
sleep 4
log_action_msg "Will now halt"
halt -d -f $netdown $poweroff $hddown
sleep 4
之后关机再也没出过磁盘响的情况了，副作用是有时候关机会稍慢一点。

3 删除vim-tiny并安装vim-full

删除tiny vim：

sudo apt-get remove vim-common
安装full vim：
sudo apt-get install vim

4 清除更新缓存

sudo apt-get clean
sudo apt-get update #------更新源
sudo apt-get upgrade #------更新已安装的包
sudo apt-get dist-upgrade # ---------升级系统
sudo apt autoremove

塞尔达传说：梦见岛DX 攻略

2013年9月18日

这份攻略是从本人（yi）2011年6月发表于levelup. im论坛（现已与xiaot论坛合并）上的图文流程攻略改编而来。流程攻略解说 部分未做大方向上的改动。只修正了一些错误并添加了说明要点部分的截图。 在此之上，还对于我所了解的部分都进行了追加和说明。希望都是正确并完善的。 攻略最终输出成了PDF文档。由于自己版面编排技巧尚不成熟，在此也希望大 家用PDF的双联显示方式来阅读本攻略。效果应该会好一些。 注：文中所使用的插画是网络上收集而来，据说是大师寺田克也早期给塞尔达 系列制作的插图。

游戏的流程部分推荐大家结合地图来进行查看。因为游戏本身的关系，每 一个场景都被分成了一格地图。所以配合地图坐标来描述、寻找地点是相对方 便的一个办法。所以流程里的地点描述我都使用了坐标。“【X-Y】”号内的 是世界地图坐标，普通不带方块符的都是相应迷宫内的坐标。在此特别说明一 下这个问题。
