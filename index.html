排序算法

2018年3月20日

0 冒泡排序

假如有一列数，我们目的是要将其递增排列。先从第一个数a开始，比较这个数和相邻的数b的大小，如果a大于b， 则交换a和b的位置（否则不交换）；接下来比较a和此刻与其相邻的数c，如果a大于c，则交换 a和c的位置......重复执行直到完成排序。

1 选择排序

假如有一列数，我们目的是要将其递增排列。先从第一个数a开始，将a设定为初始最小值，接着遍历后面的数， 如果碰到一个比a更小的数，将其设定为新的最小值，接着遍历后面的数，直到后面没有比其更小的数（如果有就又将其设定为新的最小值）， 交换新的最小值和初始最小值的位置。第二次将第二个数设定为初始最小值......重复上述操作完成排序。

2 插入排序

假如有一列数，我们目的是要将其递增排列。先从第一个数a开始，将a设定为已排过序的数，提取第一个未排序的数b， 将b和它前一个已排过序的数（此时为a）进行比较，如果a大于b则交换a和b的位置并插入空位（a之前的位置），此刻顺序为b,a,c,... 然后提取此时状态下第一个未排序的数c，将其与前面排过序的所有数（此刻为b和a）依次比较大小：若c小于a则交换两者的位置，再比较c和b的 大小，若b小于c，则将c插入空位（a刚刚的位置），此刻顺序为b,c,a,d,e,...提取d.....重复上述操作完成排序。

3 归并排序

假如有一列数，我们目的是要将其递增排列。将所有的数分散各自成组。将第一个组和第二组最左侧的数进行比较，申请一个新的数组，大小为刚 好能装下第一组和第二组，若第一组最左侧的数比第二组最左侧的数小，则将最小的复制到新数组里面，然后复制较大的数到新数组里，此刻新数组里 的顺序为从小到大，将新数组看作整体，复制回（嵌入）原来的数组中，将新数组与后一个数组进行相似的比较......完成排序。

4 快速排序

假如有一列数，我们目的是要将其递增排列。快速排序的思想是随机选择第一个数作为轴值，将数列中大于轴值的放右边，小于轴值的放左边 完成这个步骤后，由轴值将数列分成两组，接着考虑小于轴值的左半部分数列，从这部分数列中选择第一个数作为轴值，重复执行之前的操作 ......直到完成排序。

5 随机快速排序

假如有一列数，我们目的是要将其递增排列。随机快速排序的思想是随机选择一个轴值，接着将数列中大于轴值的放右边，小于轴值的放左边 完成这个步骤后，由轴值将数列分成两组，接着考虑小于轴值的左半部分数列，从这部分数列中随机选择一个轴值，重复执行之前的操作 ......直到完成排序。

6 计数排序

假如有一列数，我们目的是要将其递增排列。计数排序的思想是新生成一个空的有序的数组，统计原数列里每个数字出现的次数，某个数每出现一次， 新的有序数序中对应的项就增加一（计数器），遍历整个数组，最后按有序数列从小到大的顺序取出每个数即可完成排序。

7 基数排序

假如有一列数，我们目的是要将其递增排列。先创建10个桶，其编号分别为0-9，遍历原数列，按个位数字依次归到编号为0-9的桶中，完成后，从 0的桶开始取出数字，注意最先放入桶中的先取出，然后取出1桶中的数字，2桶中的数字......完成后得到新的数列，再次遍历得到的数列，这时按 十位数字依次归到编号为0-9的桶中，再从0桶开始取出数字......重复操作直到完成排序。

Ubuntu 16.04已知问题集锦

2017年12月7日

0 Nvidia显卡安装

sudo apt-get install nvidia-384

1 亮度调节问题

修改/etc/default/grub：

sudo gedit /etc/default/grub
将其中的
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"
改为
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash acpi_backlight=vendor"
然后更新grub：
sudo update-grub
重启后在/sys/class/backlight/下多了一个ideapad的文件夹， /sys/class/backlight/ideapad目录下的brightness和actual_brightness保存 了当前屏幕亮度值，max_brightness是屏幕最大亮度。此时即可调节亮度。

2 关机硬盘异响

修改/etc/init.d/halt的文件：

sudo gedit /etc/init.d/halt
里面有个stop函数，有几行如下
log_action_msg "Will now halt"
halt -d -f $netdown $poweroff $hddown
将其改为
umount -a
sleep 4
log_action_msg "Will now halt"
halt -d -f $netdown $poweroff $hddown
sleep 4
之后关机再也没出过磁盘响的情况了，副作用是有时候关机会稍慢一点。

3 删除vim-tiny并安装vim-full

删除tiny vim：

sudo apt-get remove vim-common
安装full vim：
sudo apt-get install vim

4 清除更新缓存

sudo apt-get clean
sudo apt-get update #------更新源
sudo apt-get upgrade #------更新已安装的包
sudo apt-get dist-upgrade # ---------升级系统
sudo apt autoremove

这份攻略是从本人（yi）2011年6月发表于levelup. im论坛（现已与xiaot论坛合并）上的图文流程攻略改编而来。流程攻略解说 部分未做大方向上的改动。只修正了一些错误并添加了说明要点部分的截图。 在此之上，还对于我所了解的部分都进行了追加和说明。希望都是正确并完善的。 攻略最终输出成了PDF文档。由于自己版面编排技巧尚不成熟，在此也希望大 家用PDF的双联显示方式来阅读本攻略。效果应该会好一些。 注：文中所使用的插画是网络上收集而来，据说是大师寺田克也早期给塞尔达 系列制作的插图。
