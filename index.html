排序算法

2018年3月20日

0 冒泡排序

假如有一列数，我们目的是要将其递增排列。先从第一个数a开始，比较这个数和相邻的数b的大小，如果a大于b， 则交换a和b的位置（否则不交换）；接下来比较a和此刻与其相邻的数c，如果a大于c，则交换 a和c的位置......重复执行直到完成排序。

1 选择排序

假如有一列数，我们目的是要将其递增排列。先从第一个数a开始，将a设定为初始最小值，接着遍历后面的数， 如果碰到一个比a更小的数，将其设定为新的最小值，接着遍历后面的数，直到后面没有比其更小的数（如果有就又将其设定为新的最小值）， 交换新的最小值和初始最小值的位置。第二次将第二个数设定为初始最小值......重复上述操作完成排序。

2 插入排序

假如有一列数，我们目的是要将其递增排列。先从第一个数a开始，将a设定为已排过序的数，提取第一个未排序的数b， 将b和它前一个已排过序的数（此时为a）进行比较，如果a大于b则交换a和b的位置并插入空位（a之前的位置），此刻顺序为b,a,c,... 然后提取此时状态下第一个未排序的数c，将其与前面排过序的所有数（此刻为b和a）依次比较大小：若c小于a则交换两者的位置，再比较c和b的 大小，若b小于c，则将c插入空位（a刚刚的位置），此刻顺序为b,c,a,d,e,...提取d.....重复上述操作完成排序。

3 归并排序

假如有一列数，我们目的是要将其递增排列。将所有的数分散各自成组。将第一个组和第二组最左侧的数进行比较，申请一个新的数组，大小为刚 好能装下第一组和第二组，若第一组最左侧的数比第二组最左侧的数小，则将最小的复制到新数组里面，然后复制较大的数到新......

Ubuntu 16.04已知问题集锦

2017年12月7日

0 Nvidia显卡安装

sudo apt-get install nvidia-384

1 亮度调节问题

修改/etc/default/grub：

sudo gedit /etc/default/grub
将其中的
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"
改为
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash acpi_backlight=vendor"
然后更新grub：
sudo update-grub
重启后在/sys/class/backlight/下多了一个ideapad的文件夹， /sys/class/backlight/ideapad目录下的brightness和actual_brightness保存 了当前屏幕亮度值，max_brightness是屏幕最大亮度。此时即可调节亮度。

2 关机硬盘异响

修改/etc/init.d/halt的文件：

sudo gedit /etc/init.d/halt
里面有个stop函数，有几行如下
log_action_msg "Will now halt"
halt -d -f $netdown $poweroff $hddown
